\# AI Server

Dedicated local AI server used for hosting large language models and developing a Jarvis-style AI assistant.

| Component | Specification |

|------------|------------|

| GPU | AMD RX 570 4 GB |

| Storage | 128 GB NVMe SSD |

/- Ubuntu Server

/- Ollama

/- Open WebUI
