\# AI Server



\## Overview



Dedicated local AI server used to host language models, speech systems, and automation services for the Jarvis AI Assistant project.



The server provides a self-hosted environment for AI experimentation, voice interaction, automation, and future multi-agent development while remaining entirely under local control.
