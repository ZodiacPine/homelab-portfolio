\# Chris Frise's Homelab Portfolio



\## Overview



This repository documents my homelab projects, infrastructure, and ongoing work in systems engineering, automation, AI infrastructure, and self-hosted services.



The primary goal of this homelab is to gain hands-on experience with Linux administration, virtualization, networking, automation, and AI systems while building a fully self-hosted Jarvis-style AI assistant.
